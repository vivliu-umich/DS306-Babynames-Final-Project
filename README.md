# Gender-Neutral Babynames Web Application
This web app explores the trends of gender-neutral names over time in the United States. Users can view trends graphically, through significance testing, or through a sorted table containing each state.

## Features
- Trend visualization for each US State
- Real-time significance test calculations
- State rankings based on gender-neutral naming percentages (2025 only)
- Neatly organized Shiny web application

## How to Run
1. Install and load the necessary packages into RStudio from the script
2. Run the app.R in RStudio
3. The Shiny app will open in a new window

## Data
The baby name data is sourced from the United States Social Security Administration (SSA).
